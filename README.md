# ProjeTask Frontend

ProjeTask Frontend es la interfaz de usuario de una aplicación de gestión de proyectos y tareas diseñada para facilitar la organización y el seguimiento de proyectos. Esta interfaz se integra con el backend del proyecto y proporciona una experiencia de usuario intuitiva y amigable.

## Descripción General

La interfaz de usuario de ProjeTask Frontend está construida con React, una biblioteca JavaScript de código abierto utilizada para construir interfaces de usuario interactivas y eficientes, junto con CSS puro para estilos y diseño. Se utiliza Axios para realizar solicitudes HTTP al backend y se hace uso de la Context API y Reductores (Reducers) para gestionar el estado global de la aplicación. Además, React Router se implementa para la navegación dentro de la aplicación, permitiendo la gestión de rutas y la transición suave entre diferentes vistas.

## Instalación y Configuración
Sigue estos pasos para configurar y ejecutar la interfaz de usuario en tu entorno local:
1. Clonar el repositorio:
```bash
git clone https://github.com/marvin211/ProjeTask-frontend.git
```

2. Navega hasta el directorio del proyecto:
```bash
cd react-ecommerce
```

3. Para instalar las dependencias del proyecto, ejecutar el siguiente comando:
```bash
npm install
```

## Uso
1. Inicia el servidor de desarrollo:
```bash
npm run dev
```

2. Abre el navegador y navega a la dirección [http://127.0.0.1:5173/](http://127.0.0.1:5173/)
